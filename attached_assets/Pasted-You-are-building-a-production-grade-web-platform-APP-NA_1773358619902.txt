IMPORTANT CONTEXT

The cloud IDE is ONLY the development environment.

All code must be portable and deployable to a standard Linux server.

The intended production environment is:

DigitalOcean Ubuntu VM
Nginx reverse proxy
Node.js backend
PostgreSQL database
PM2 process manager
HTTPS via Let's Encrypt

The application must be fully portable and deployable outside the development IDE.

--------------------------------------------------

PROJECT PURPOSE

GNNcab is a secure multi-city ride dispatch and driver tracking platform designed for local transportation networks.

Initial live deployment may start small, but the architecture must be designed for large scale expansion.

Design the system so it can evolve to support:

500+ cities
thousands of zones
10,000+ drivers
high trip volumes
horizontal scaling
separation of services in the future

Initial deployment may be simple.
Architecture must be scalable by design.
